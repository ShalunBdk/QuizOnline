# Quiz Platform

Интерактивная платформа для проведения квизов в реальном времени (в стиле Kahoot).

## Возможности

- 🎮 Создание квизов с несколькими вариантами ответов
- 🔴 Работа в реальном времени через WebSocket
- 📊 Система подсчета очков с учетом скорости ответа
- 📺 Отдельный экран для проектора
- 📱 QR-код для быстрого подключения
- 🏆 Таблица лидеров

## Запуск с Docker Compose

### Требования
- Docker
- Docker Compose

### Быстрый старт

1. Клонируйте репозиторий или перейдите в папку проекта

2. Запустите приложение:
```bash
docker-compose up -d
```

3. Откройте в браузере:
   - Организатор: http://localhost:3000/host
   - Участники: http://localhost:3000/play
   - Экран презентации: http://localhost:3000/screen

### Управление контейнером

Остановить:
```bash
docker-compose down
```

Перезапустить:
```bash
docker-compose restart
```

Посмотреть логи:
```bash
docker-compose logs -f
```

Пересобрать после изменений:
```bash
docker-compose up -d --build
```

## Запуск без Docker

### Требования
- Node.js 18+
- npm

### Установка и запуск

```bash
npm install
npm start
```

Приложение будет доступно на http://localhost:3000

## Как использовать

1. **Создайте квиз** - откройте `/host` и добавьте вопросы
2. **Создайте сессию** - получите код для подключения
3. **Подключите участников** - они заходят на `/play` и вводят код
4. **Подключите экран** (опционально) - откройте `/screen` на проекторе
5. **Начните квиз** - нажмите "Начать квиз" в панели организатора
