# Quiz Platform

Интерактивная платформа для проведения квизов в реальном времени (в стиле Kahoot).

## Возможности

- 🎮 Создание квизов с несколькими вариантами ответов
- 🔗 Уникальная ссылка для каждого квиза (приватное редактирование)
- 💾 Автоматическое сохранение в SQLite
- 📊 Редактор финального опроса с разными типами вопросов
- 💬 Сохранение результатов опросов для анализа
- 🔀 Перемешивание вариантов ответов для каждого участника
- 🔴 Работа в реальном времени через WebSocket
- 📈 Система подсчета очков с учетом скорости ответа
- 📺 Отдельный экран для проектора
- 📱 QR-код для быстрого подключения
- 🏆 Таблица лидеров

## Запуск с Docker Compose

### Требования
- Docker
- Docker Compose

### Быстрый старт

1. Клонируйте репозиторий или перейдите в папку проекта

2. Запустите приложение:
```bash
docker-compose up -d
```

3. Откройте в браузере:
   - Организатор: http://localhost:3000/host
   - Участники: http://localhost:3000/play
   - Экран презентации: http://localhost:3000/screen

### Управление контейнером

Остановить:
```bash
docker-compose down
```

Перезапустить:
```bash
docker-compose restart
```

Посмотреть логи:
```bash
docker-compose logs -f
```

Пересобрать после изменений:
```bash
docker-compose up -d --build
```

## Запуск без Docker

### Требования
- Node.js 18+
- npm

### Установка и запуск

```bash
npm install
npm start
```

Приложение будет доступно на http://localhost:3000

## Как использовать

### Создание квиза

1. **Откройте главную страницу** - перейдите на http://localhost:3000
2. **Нажмите "Создать квиз"** или откройте `/host`
3. **Нажмите "📝 Новый квиз"** - будет создан квиз с уникальной ссылкой
4. **Сохраните ссылку!** - она понадобится для редактирования
5. **Заполните информацию**:
   - Введите название квиза
   - Добавьте описание (необязательно)
   - Создайте вопросы с вариантами ответов
   - Настройте время и перемешивание
6. **Нажмите "💾 Сохранить"** - квиз сохранится в БД

### Редактирование квиза

1. **Откройте сохранённую ссылку** - например `/host?quiz=quiz_abc123`
2. **Внесите изменения**
3. **Нажмите "💾 Сохранить"**

### Проведение квиза

1. **Откройте квиз по ссылке** - `/host?quiz=quiz_abc123`
2. **Нажмите "🚀 Создать сессию"**
3. **Откройте экран для зрителей** (опционально):
   - Нажмите кнопку "📺 Открыть экран для зрителей" в лобби
   - Или откройте `/screen` вручную и введите код
4. **Подключите участников** - они заходят на `/play` и вводят код
5. **Начните квиз** - нажмите "▶ Начать квиз"

Во время квиза кнопка "📺 Экран" доступна на всех этапах для быстрого открытия экрана презентации.

### Важно!

- Каждый квиз имеет уникальную ссылку для редактирования
- Только тот, у кого есть ссылка, может редактировать квиз
- Сохраните ссылку в закладки или отправьте себе
- Квизы хранятся в БД и доступны по ссылке всегда
