# Руководство по опросам

## Создание опроса

1. Откройте редактор квиза
2. Найдите секцию "📊 Финальный опрос"
3. Нажмите "Настроить"
4. Нажмите "+ Добавить вопрос"

## Типы вопросов

### 1. Оценка (звёзды)
- Участники ставят оценку от 1 до 5 звёзд
- Результат: средняя оценка
- Пример: "Насколько полезен был материал?"

### 2. Множественный выбор
- Участники выбирают один или несколько вариантов
- **Важно**: Каждый вариант на новой строке!
- Результат: количество выборов каждого варианта

**Как добавлять варианты:**
```
Вариант 1
Вариант 2
Вариант 3
```

**НЕ ТАК:**
```
Вариант 1, Вариант 2, Вариант 3
```

### 3. Текстовый ответ
- Участники пишут свободный текст
- Результат: список всех ответов
- Пример: "Какие темы хотели бы изучить?"

## Проведение опроса

1. Завершите квиз
2. На экране появится кнопка "📋 Запустить опрос"
3. Участники увидят вопросы опроса
4. После ответов нажмите "Показать результаты"

## Просмотр результатов

### Во время сессии
- Нажмите "Показать результаты" после опроса
- Результаты отобразятся на экране

### После закрытия сессии
1. Откройте квиз по ссылке
2. Перейдите на вкладку "Результаты"
3. Увидите список всех проведённых опросов
4. Нажмите "Показать результаты" для просмотра

## Результаты сохраняются автоматически

- При нажатии "Показать результаты" данные сохраняются в БД
- Привязаны к конкретному квизу
- Хранятся с датой и кодом сессии
- Доступны для просмотра в любое время

## Советы

- Создавайте опрос заранее вместе с квизом
- Используйте понятные формулировки вопросов
- Для множественного выбора: 3-5 вариантов оптимально
- Текстовые вопросы используйте для открытой обратной связи
