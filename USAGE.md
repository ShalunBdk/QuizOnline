# Как использовать Quiz Platform

## Создание нового квиза

1. Откройте http://localhost:3000
2. Нажмите "🎮 Создать квиз"
3. Нажмите кнопку "📝 Новый квиз"
4. **ВАЖНО!** Скопируйте и сохраните ссылку для редактирования
   - Она выглядит так: `http://localhost:3000/host?quiz=quiz_abc123xyz`
   - Только с этой ссылкой можно редактировать квиз
   - Сохраните её в закладки или отправьте себе

## Создание вопросов

1. Введите название квиза
2. Добавьте описание (необязательно)
3. Нажмите "+ Добавить вопрос" для каждого нового вопроса
4. Для каждого вопроса:
   - Введите текст вопроса
   - Заполните варианты ответов (минимум 2)
   - Отметьте правильный ответ (кликните на кружок)
   - Добавьте пояснение (необязательно)

## Настройки

- **Время на вопрос** - сколько секунд даётся на ответ (5-120)
- **Перемешивать варианты** - каждый участник видит варианты в случайном порядке

## Сохранение

Нажмите "💾 Сохранить" - квиз сохранится в базу данных

## Редактирование существующего квиза

1. Откройте сохранённую ссылку: `http://localhost:3000/host?quiz=quiz_abc123xyz`
2. Квиз загрузится автоматически
3. Внесите изменения
4. Нажмите "💾 Сохранить"

## Проведение квиза

1. Откройте квиз по ссылке
2. Нажмите "🚀 Создать сессию"
3. Появится QR-код и код для входа
4. Участники:
   - Заходят на http://localhost:3000/play
   - Вводят код
   - Вводят своё имя
5. (Опционально) Откройте экран для проектора:
   - http://localhost:3000/screen
   - Введите тот же код
6. Когда все подключились, нажмите "▶ Начать квиз"

## Управление во время квиза

- Вопросы переключаются автоматически по таймеру
- Можно завершить вопрос досрочно кнопкой "Завершить вопрос"
- После каждого вопроса показываются результаты
- В конце - таблица лидеров

## Безопасность

- Каждый квиз имеет уникальный ID в ссылке
- Без ссылки невозможно найти или отредактировать квиз
- Не делитесь ссылкой для редактирования с теми, кому не доверяете
- Для участников используйте только код сессии, а не ссылку на квиз

## Советы

- Создавайте резервные копии важных квизов (экспорт JSON)
- Тестируйте квиз перед реальным использованием
- Для больших аудиторий используйте экран презентации
- Перемешивание ответов помогает избежать списывания
